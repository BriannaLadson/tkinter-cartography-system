# tkinter-cartography-system
A sandbox-style cartography simulation written in Python and Tkinter.

This project is a vertical slice exploring how cartography can function as a gameplay system rather than a simple map screen.

***

## Current Features
* Random world generation
* Discoverable locations
* Zoomable tile map
* NPC cartographers
* Cartography skill and experience system
* Map creation
* Skill-based map accuracy
* Buying and selling maps
* In-game calendar and time progression
* Location discovery tracking

***

## Controls
* Arrow Keys | Move
* 1-9 Numpad | Diagonal Movement
* (+) Plus Sign | Zoom In
* (-) Minus Sign | Zoom Out
* 5 | Open Shop (Town/Village Only)

***
